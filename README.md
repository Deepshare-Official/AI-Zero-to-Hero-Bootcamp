## 📖 本书简介

本项目是《人工智能0基础通关训练营》的官方开源配套仓库。本书采用项目驱动教学模式，通过 21 个工业级案例串联起 Python 编程、数据处理、机器学习、PyTorch 框架及深度学习算法。

书中所独创的“三阶成长路径”旨在帮助零基础学习者在 12 周内掌握计算机视觉、自然语言处理和推荐系统等领域的 34 种主流算法，真正实现“学完即用，用即见效”。

## 🛤️ 三阶成长路径

### 🏗️ 阶段一：编程技能篇
通过实战案例自然引入 AI 开发必备的 Python 核心库（Numpy, Pandas, Matplotlib），告别枯燥语法，打牢 AI 编程根基。

### 🧠 阶段二：机器学习篇
系统介绍主流算法及实现库，穿插必要的数学基础。适合希望从原理到应用全面掌握机器学习的开发者。

### 🕸️ 阶段三：深度学习篇
基于 PyTorch 框架，深度解析 **CV (计算机视觉)**、**NLP (自然语言处理)** 及 **推荐系统** 三大领域的基石算法。

## 🛠️ 如何使用本项目

1. **环境克隆**：建议使用 `conda` 创建虚拟环境。
2. **代码运行**：每个章节对应的 `Notebook` 均包含完整的数据预处理和模型训练代码。
3. **互动交流**：如果在运行中遇到 Bug 或有改进建议，请直接提交 **Issue**。
