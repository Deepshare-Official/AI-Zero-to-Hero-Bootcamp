## 📖 项目简介

本项目是《人工智能0基础通关训练营》的官方开源配套仓库。本书是一本以项目驱动为核心的 AI 入门指南。

我们致力于突破传统技术教材的枯燥模式，通过 **21 个工业级实战案例**串联起人工智能的完整知识体系。书中独创了“三阶成长路径”设计，旨在帮助零基础学习者在 12 周内完成从编程小白到 AI 开发者的蜕变，熟练掌握 **34 种主流深度学习算法**在计算机视觉、自然语言处理和推荐系统中的实际应用。

仓库内配套提供完整的项目代码，真正实现“学完即用，用即见效”的学习目标。

## 🛤️ 三阶成长路径 (核心内容)

强烈建议学习者严格按照书中的章节顺序进行学习，切忌跳跃，以达到最佳效果：

### 🧱 阶段一：编程技能篇
万丈高楼平地起。本篇并非枯燥的语法罗列，而是通过精心设计的实战案例，自然地穿插讲解 AI 开发必需的 Python 核心概念库（如 Numpy, Pandas, Matplotlib）及关键技能，帮你打牢 AI 编程根基。

### 🧠 阶段二：机器学习篇
正式踏入 AI 核心领域。本篇系统介绍了当下最热门的算法及其主流实现库，并在学习过程中适时穿插背后的数学基础和概念，确保 0 基础也能顺利理解和实践。

### 🕸️ 阶段三：深度学习篇
从当前主流的 PyTorch 框架入手，深度剖析三大黄金应用领域：**计算机视觉（CV）、自然语言处理（NLP）和推荐系统**。你将在实战项目中掌握 CNN、RNN、Transformer 等基石算法的运作机制。

## 🛠️ 食用指南

1. **动手实践**：本书的精髓在于“做中学”。请务必认真完成配套的编程练习，尝试复现和修改代码。
2. **一键运行**：本项目提供了开箱即用的 Jupyter Notebook 代码及数据集，请参考目录下的 `环境配置指南` 进行基础配置。
3. **主动思考**：在学习算法时，多对比不同方法的适用场景与优劣，形成自己的知识图谱。
